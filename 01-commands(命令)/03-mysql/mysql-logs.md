| 操作目的  | SQL 命令     | 说明    |

|---|---|---|

| 查询错误日志文件路径 | `SHOW VARIABLES LIKE 'log\_error';` | 获取 MySQL 错误日志文件的路径 |

| 查询通用日志（查询日志）文件路径 | `SHOW VARIABLES LIKE 'general\_log\_file';` | 获取通用日志（记录所有语句）的路径   |

| 查询慢查询日志文件路径 | `SHOW VARIABLES LIKE 'slow\_query\_log\_file';`      | 获取慢查询日志文件的路径             |

| 查看二进制日志（Binary Logs）    | `SHOW BINARY LOGS;` | 列出当前服务器上的二进制日志文件列表 |

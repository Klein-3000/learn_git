# 基本用法
## base
```
# 命令格式
obsidian <subcommand> [file]{名字即可，不需要**md**后缀} | [path]{主要是解决不同目录下同名文件的问题}

# 查看帮助
obsidian help <subcommand>
```
## read
```
obsidian read file="<name>"

```
## move
```
obsidian move file=<name> to=<path>
```
## rename
```
obsidian rename file=<name> name=<name>
```
## outline(大纲)
```
obsidain outline file=<name> format=tree | md | json
```

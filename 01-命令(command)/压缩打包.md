
| 命令  | tar                                            | zip                           | unzip                                               |
| --- | ---------------------------------------------- | ----------------------------- | --------------------------------------------------- |
| 压缩  | tar -zcf \[path/]file.tar.gz  old_File         | zip \[path/]file.zip old_File |                                                     |
| 查看  | tar -tvf \[path/]file.tar.gz                   |                               | unzip -lv \[path/]file.zip                          |
| 解压  | tar -zxf \[path/]file.tar.gz \[-C path/to/dir] |                               | unzip \[path/]file.zip \[-d path/to/dir] \[-x "正则"] |

# nmap
## 扫描存在的主机
```bash
# 指定ip范围
nmap -sn 192.168.1.0-100

# 指定网段
nmap -sn 192.168.1.0/24
```

| 命令                    | 作用                   |
| --------------------- | -------------------- |
| nmap -O \<ip>         | 探测系统(OS)             |
| nmap \<ip>            | 扫描ip(显示端口,服务)        |
| nmap -sV \<ip>        | 查看服务的版本(显示端口,服务，+版本) |
| nmap -p n1,n2.. \<ip> | 扫描指定的端口              |

# 一、基本使用
文字聊天
```shell
# server
nc  [-d]{daemon : 后台运行} [-l]{listen: 侦听} [-p]{port : 端口} <localport>

# client
# w(wait)参数,避免假死
nc [-w]{wait : 等待}2 <remote_ip> <remote_port>

 
```
## 返回shell
```shell
# server
nc -l -p <localport> -e < cmd.exe | pwsh.exe | /bin/bash >

 
```
> [!attention] 注意
> 默认在`nc`server 执行所在的目录

# 二、文件传输
文字聊天的进阶,通过重定向,一次性发送或接收大量**文字**
```shell
# client --> server
# server(receiver) 接收文件
nc -l -p <localport> [>]{**输出**重定向} <outfile>

# client(sender) 传输文件
nc <remote_ip> <remote_port> [<]{**输入**重定向} <infile>
 
```
> [!attention] 注意
> 通过**重定向**
> 可以是 server --> client
> 也可以 client --> server
# 三、文件夹传输
```shell
# client --> server
# server(receiver) 接收文件
nc -l -p <localhost> | tar -x

# client(sender) 传输文件
tar -cf [-]{允许传输大文件} <dir> | nc <remote_ip> <remote_port>
 
```
> [!attention] 注意
> `tar` 在**前**为==上传==
> `tar` 在**后**为==下载==
